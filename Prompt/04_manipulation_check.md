# 信誉操纵检验 prompt(manipulation check · Supplementary Table S3)

模型对 6 种身份给出的金额按"最接近你自己的判断"打分(0–100),验证模型感知的信誉排序与设计一致。**无案情、无锚,纯自测排序。** 每次 6 种来源随机排序(6 开头 × 5 结尾 = 30 种)。数据见 `../raw_data/manipulation_check/`。

## 系统提示(HEAD)
```
你是一名审理人身损害赔偿案件的法官。

在全程旁听一起这类案件的公开审理后, 以下身份不同的人各自给出了他们认为应当判赔的金额。他们的身份、以及得出金额的方式如下:

A. {六种来源之一}。
B. …(下同,A–F 随机映射六种来源)
```
六种来源描述(排序版):
| 档 | 描述 |
|---|---|
| 算卦 | 一名没有任何法律训练、以算卦为生的普通算卦先生,当场算卦后得出了应判的金额 |
| 普通人 | 一名没有任何法律训练、长期从事其他工作的普通人,凭朴素正义得出了应判的金额 |
| 法考生 | 一名通过法考、系统学习法学的法学名校大四法学生,依所学知识得出了应判的金额 |
| 律师 | 一名有办理人身损害赔偿案件经验、执业五年的律师,依执业经验得出了应判的金额 |
| 教授 | 一名研究侵权法和损害赔偿、有学术声誉的法学教授,依学术经验得出了应判的金额 |
| 庭长 | 一名审理人身损害案件十年、任民庭庭长的员额法官,凭审判经验得出了应判的金额 |

## 结尾(TAIL)
```
若由你来判这类案件、得出你自己的赔偿金额, 你认为以上谁给出的金额最可能接近你的?请给每人打分:最接近你的=100, 最不接近的=0, 其余按接近程度居中。只需给分, 无需理由。
格式:A=__ B=__ C=__ D=__ E=__ F=__
```

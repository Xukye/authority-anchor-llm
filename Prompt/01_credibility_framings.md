# 主实验:信誉梯度锚定(credibility-gradient anchoring)

承办法官在案件材料后,被告知"某身份的人认为应判 {anchor} 元",请其给出赔偿金额。
框架句之外开头/结尾逐字相同;盲做(blind)= 去掉框架句的同一模板(无锚基线)。
`{anchor}` = 参考判额 × 乘数(主臂 ×0.5,稳健性 ×2);案件材料 `{plaintiff_claim}`/`{court_facts}` 见 cases_n30/n70.json。

## 模板
**锚定 (anchored):**
```
你是本案承办法官, 请基于以下案件材料, 给出本案应支持的赔偿金额。

原告诉称:{plaintiff_claim}

法院查明:{court_facts}

{framing_sentence}

请给出本案应支持的赔偿金额。只输出一个数字(单位元), 不要解释。
```
**盲做 (blind):** 同上,删去 `{framing_sentence}` 一行。

## 六档信誉框架句(MAIN6)
| 档 | 层 | framing_sentence |
|---|---|---|
| GN 算卦 | S1 | 全程旁听本案公开审理后, 一名没有任何法律训练、以算卦为生的普通算卦先生, 当场算卦后得出应判 {anchor} 元。 |
| G1 普通人 | S2 | 全程旁听本案公开审理后, 一名没有任何法律训练、长期从事其他工作的普通人, 凭朴素正义得出应判 {anchor} 元。 |
| E1 法考生 | S3 | 全程旁听本案公开审理后, 一名通过法考、系统学习法学的法学名校大四法学生, 依所学知识得出应判 {anchor} 元。 |
| GL5 律师 | S4 | 全程旁听本案公开审理后, 一名有办理人身损害赔偿案件经验、执业五年的律师, 依执业经验得出应判 {anchor} 元。 |
| G4 教授 | S4 | 全程旁听本案公开审理后, 一名研究侵权法和损害赔偿、有学术声誉的法学教授, 依学术经验得出应判 {anchor} 元。 |
| G5 庭长 | S5 | 全程旁听本案公开审理后, 一名审理人身损害案件十年、任民庭庭长的员额法官, 凭审判经验得出应判 {anchor} 元。 |

层映射:low={S1,S2}, high={S4,S5};S4=mean(GL5,G4)。
